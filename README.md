# Copy Board — 常用語收藏

簡介
---
Copy Board 是一個小型前端工具，把常用文字片段存在瀏覽器 localStorage，方便隨時複製、編輯、匯入/匯出 JSON 備份。

新增 UI 功能
---
- 排序（Sort）：可選「收藏優先」、「依新增時間（最新/最舊）」、「字母排序（A→Z / Z→A）」。
- 標籤快速篩選（Tag filters）：搜尋框下方會自動列出所有標籤，點擊標籤可快速篩選（可多選，任一 tag 符合即會顯示）。
- 清除標籤：可使用「清除標籤」按鈕清空已選標籤。
- 收藏（favorite）功能維持：收藏項目會優先顯示（排序選擇會再基於此做第二排序）。

主要功能
---
- 新增、編輯（Modal）、刪除項目
- 收藏 / 取消收藏（小星號）
- 複製到剪貼簿（Clipboard API）
- 搜尋（文字與標籤）
- 標籤快速點擊篩選（在搜尋框下方）
- 排序（Sort）
- 匯出 JSON、匯入 JSON（匯入會 prepend 到現有列表）
- 清除所有
- 項目存在 localStorage（key: `copyBoard.items`）

使用說明
---
1. 把檔案放同一個資料夾：`index.html`、`app.js`、`style.css`，然後在瀏覽器開啟 `index.html`。
2. 搜尋：輸入文字或標籤會即時過濾。
3. 點擊搜尋下方的標籤可快速篩選（可點多個），點擊「清除標籤」移除所有 tag 篩選。
4. 在排序選單選擇你要的排序方式；「收藏優先」是預設。
5. 按「新增」新增新條目，標籤用逗號分隔。
6. 按每個卡片的星號標記收藏，收藏的項目會自動排到列表上方（依排序規則）。
7. 按「修改」可在彈窗編輯內容與標籤，按「儲存變更」後會更新並存回 localStorage。
8. 按「複製」會把內容放到剪貼簿並短暫變色提示。
9. 匯出會產生 JSON 檔案，匯入接受陣列格式的 JSON，並保留 favorite 欄位。

範例匯入 JSON
```json
[
  {"id":"abc","text":"範例文字","tags":["a","b"],"favorite":true,"createdAt":"2025-01-01T00:00:00Z"}
]
```
