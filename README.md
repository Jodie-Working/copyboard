# Copy Board（標籤 + 置頂 + LocalStorage + 匯入/匯出）

一個簡易嘅常用語 Copy Board：
- 標籤（多個 tags）
- 置頂（加星號）
- LocalStorage 儲存（瀏覽器內儲存）
- 匯入/匯出 JSON（備份/跨裝置同步）
- 搜尋、標籤篩選、一鍵複製

## 使用方法
1. 打開網站（GitHub Pages 或直接本地 double click `index.html`）。
2. 在「新增詞條」輸入：
   - 文字（必填）
   - 標籤（選填，逗號分隔）
3. 點擊詞條上的：
   - 星號：置頂/取消置頂
   - 複製：寫入剪貼簿
   - 修改：用對話框快速改文字/標籤
   - 刪除：移除該詞條
4. 搜尋框支援即時篩選；可按標籤按鈕疊加篩選。
5. 匯出 JSON：下載目前所有詞條（含置頂、時間、標籤）。
6. 匯入 JSON：選擇備份檔，合併或取代現有資料。

## JSON 格式
匯出/匯入的結構如下（可手動編輯）：
```json
[
  {
    "id": "uuid-xxxx",
    "text": "收到，謝謝！",
    "tags": ["工作", "回覆"],
    "pinned": true,
    "createdAt": 1731800000000
  }
]
